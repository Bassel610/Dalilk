# Dalilk

Full-stack web application — React frontend with an Express + Firebase backend, audio recording, and PDF report generation.

## Tech Stack

- **Frontend:** React 18, React Router, Axios
- **Backend:** Node.js, Express, Firebase
- **Other:** jsPDF + jspdf-autotable for PDF reports, react-mic / @saassy/audio-react-recorder for audio capture

## Project Structure

```
Dalilk/
├── My-app/        # React frontend
└── My-backend/    # Express server
```

## Getting Started

```bash
# Frontend
cd My-app
npm install
npm start

# Backend
cd My-backend
npm install
npm start
```
